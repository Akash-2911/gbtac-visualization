# GBTAC Energy Visualization

A web application that transforms greenhouse energy monitoring data into interactive dashboards and AI-powered insights.

---

## The Story

GBTAC operates a net-zero greenhouse in Alberta, Canada. The building runs on concentrated solar energy, heat pumps, and passive solar design. Sensors record energy consumption, solar generation, and outdoor weather conditions continuously.

All of this data was sitting in Excel files with no easy way to visualize trends, track net-zero performance, or share results with stakeholders.

This project changes that.

---

## What We Built

A full-stack web application that turns raw sensor data into something anyone can understand and use.

**Six interactive dashboards** covering energy consumption by system, solar generation, CO2 emissions, weather vs building performance, side-by-side energy comparison, and a live overview with key metrics.

**Three AI-powered features** — a natural language chat assistant, automated performance summaries, and predictive energy forecasting.

**Admin panel** for uploading new datasets with automated validation and processing.

**Role-based access** for Admin, Staff, and Viewer users — all authenticated via Microsoft Entra ID.
